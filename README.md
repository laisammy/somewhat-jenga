# Somewhat Jenga(?)
This is a minigame made with Godot 4.6.1. The goal of the game is to stack jenga bricks as high as you can.

# Some key features
You can position your brick by moving and rotating it. Raycasting is also used to show a black circle on where the brick is going to drop, allowing you to precisely place your brick at the correct spot. Your high score is also saved!

# Controls
WASD to move brick
QE to rotate brick
Space to drop brick

# Hardships, difficulties, problems blah blah blah
Raycasting was something I've learned while developing games but never really applied it to my projects. For this project, I spent a while learning how Godot's raycasting worked and I managed to pull it off after an hour to trying to figure out.
